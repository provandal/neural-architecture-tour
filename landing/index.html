An interactive series

Neural Architecture Tour

Build intuition for the models that power modern AI — one architecture at a time. Each episode is a Colab notebook, a browser app, and a narrative — so you learn the architecture and the tools practitioners use for it.

Notebook
Learn the mechanics in PyTorch, scikit-learn, or HF — the real tools.
Browser app
Play with a trained model. Visualize. Break things. Run inference client-side.
Curriculum
A narrative tying it all together. No glossaries — only things you can do.
